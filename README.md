# Mongolia History Timeline
An interactive, data-driven web application visualizing key events in Mongolian history.

## Features
- Category filtering (Empire, Culture, Politics, Modern)
- Live search functionality
- Dynamic event detail rendering
- Conditional image display
- Responsive layout using CSS Grid
- Themed UI with Mongolian flag-inspired design

## Tech Stack
- HTML
- CSS
- JavaScript
- JSON

## Project Purpose
Built as a side project to strengthen frontend development skills and demonstrate structured data handling, DOM manipulation, and UI design.
